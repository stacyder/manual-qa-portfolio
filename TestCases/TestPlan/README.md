Test Plan for OrangeHRM Demo Application

Scope:
Testing of main functionalities including Login, PIM, and Admin modules.

Objectives:
- Verify core functionality works correctly
- Identify defects in UI and functionality

Testing Types:
- Functional testing
- UI testing
- Exploratory testing

Test Environment:
- Browser: Chrome
- OS: macOS

Entry Criteria:
- Application is accessible
- Test data is available

Exit Criteria:
- All critical test cases executed
- Major bugs identified

Risks:
- Limited access to backend
- Demo environment instability
