Checklist for OrangeHRM

Login
- Verify login with valid credentials
- Verify login with invalid password
- Verify empty fields
- Verify login with locked user
- Verify error message display

PIM (Employee Management)
- Add new employee
- Search employee by name
- Delete employee
- Verify required fields validation
- Verify employee data is saved correctly

Admin
- Add new user
- Search user
- Assign user role
- Change user status (enabled/disabled)
- Verify user appears in list
