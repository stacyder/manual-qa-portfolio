Test Plan for OrangeHRM

Scope
Testing of core functionalities: Login, PIM (Employee Management), and Admin module.

Objectives
- Verify main features work correctly
- Identify UI and functional defects

Testing Types
- Functional testing
- UI testing
- Exploratory testing

Test Environment
- Browser: Chrome
- OS: macOS

Entry Criteria
- Application is accessible
- Test data is available

Exit Criteria
- All main test cases are executed
- Critical defects are identified

Risks
- Limited access to backend
- Possible instability of demo environment
